**Motion Visualizer** — демонстрационный прототип программного комплекса для визуализации движения объектов в трёхмерной среде с поддержкой математического моделирования и интеграции с базами данных.

Проект разработан как доказательство концепции (Proof of Concept) для последующего масштабирования до уровня промышленного цифрового полигона.

## Быстрый старт

### Требования

- Python 3.10 – 3.13
- C++17/20 компилятор (gcc 11+ / MSVC 2022)
- OpenGL 3.3+
- CMake 3.15+
- pybind11

### Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/DIK2022/motion_visualizer.git
cd motion_visualizer

# Установка uv (менеджер пакетов)
curl -LsSf https://astral.sh/uv/install.sh | sh

# Установка зависимостей Python
uv sync

# Сборка C++ модуля
mkdir -p build && cd build
cmake ..
make
cd ..

# Запуск приложения
uv run python -m motion_visualizer.main
```

После запуска откроется главное окно приложения с кнопкой «Старт» и областью 3D-визуализации. При нажатии кнопки «Старт» красный квадрат начнёт движение под управлением C++ кода.
